Initializing Card Object

// Pass the id for div tag in your HTML or the HTMLElement. In our example, we used 'card-id'.
  const card = new Card('card-id', 'card1');
Property Default Type Description
selector (required) String/HTMLElement This is the id of the div tag in the HTML or the HTMLElement that you want to display the card in.
id (required) String This is the id of the card that you will be creating.

Always use methods mentioned below to modify the cards.

Card Methods

// Create a General Card
  makeGeneralCard(title, subtitle, imgSrc)

This method creates a General Card with the given properties. The method should strictly follow the following property requirements:

Property Requirement Type Description
title (required) String This is the title of the card. This is displayed in large text in general card.
subtitle (optional) String This is the subtitle of the card. This is displayed as small text below title.
imgSrc (optional) String This is the source of the image that will be displayed in the front of the card. 
// Create a Product Card
  makeProductCard(title, desc, imgSrc, button1, button2)

This method creates a Project Card with the given properties. The method should strictly follow the following property requirements:

Property Default Type Description
title (required) String This is the title of the card. This is displayed in large text in product card.
desc (required) String This is the desc of the card. This is displayed as text below title which describes the product.
imgSrc (required) String This is the source of the image that will be displayed in the front of the card.
button1 (required) Object You need to provide the text and link for the button in an object. It is displayed in the bottom of the card.
button2 (required) String You need to provide the text and link for the button in an object. It is displayed in the bottom of the card next to button1. 
// Create a Teammate Card
  makeTeammateCard(title, subtitle, imgSrc, links)

This method creates a Teammate Card with the given properties. The method should strictly follow the following property requirements:

Property Default Type Description
title (required) String This is the title of the card. This is displayed the name of the teammate in teammate card.
subtitle (optional) String This is the subtitle of the card. This is displayed as small text below title which is used to tells what the teammate does.
imgSrc (optional) String This is the source of the image that will be displayed in the front of the card.
links (required) Object This is an object that is used to show links on a card below the subtitle with their icons.

Example link object

 {
      'github': 'https://github.com/',
      'instagram': 'https://www.instagram.com/',
      'linkedin': 'https://linkedin.com'
   }
  
// Create a Project Card
  makeProjectCard(title, desc, links)

This method creates a Project Card with the given properties. The method should strictly follow the following property requirements:

Property Default Type Description
title (required) String This is the title of the card. This is displayed in large text in project card.
desc (required) String This is the desc of the card. This is displayed as text below links which describes the project.
links (required) Object This is an object that is used to show links on a card below the title with their icons. 
// Add Zoom Feature on a card
  addZoom()

This method adds the ability to zoom to the card. It will zoom the card when you hover over it.

// Add a flip feature and show the backview of the card
  addFlip(title, desc)

This method adds the ability to flip a card. It will flip the card when you click on the card and show you in the backview of that card.

Property Default Type Description
title (required) String This is the title of the card. This is displayed in large text in backview of that card.
desc (optional) String This is the description of the card. This is displayed as small text below title in the backview.

Customizing your card according to the style

// Changes the background color
  changeBackgroundColor(color)

This method changes the color of the card. The requirement for this method is as explained below:

Property Default Type Description
color (required) String You need to provide the color (in hex, word or rbg) that you want the background to be. 
// Changes the title color
  changeTitleColor(color)

This method changes the title color of the card. The same requirement as changeBackgroundColor.

// Changes the subtitle color
  changeSubtitleColor(color)

This method changes the subtitle color of the card. The same requirement as changeBackgroundColor.

// Changes the description color
  changeDescriptionColor(color)

This method changes the description color of the card. The same requirement as changeBackgroundColor.

// Chnages the backview title color
  changeBackTitleColor(color)

This method changes the backview title color of the card. The same requirement as changeBackgroundColor.

// Chnages the back description color
  changeBackDescriptionColor(color)

This method changes the back description color of the card. The same requirement as changeBackgroundColor.

Initializing CardList Object

// Pass the id for div tag in your HTML or the HTMLElement. In our example, we used 'card-id'.
  // It creates a cardList which is screen responsive
  const card = new CardList('card-id', 'red', 'white');
Property Default Type Description
selector (required) String/HTMLElement This is the id of the div tag in the HTML or the HTMLElement that you want to display the card in.
cardColor (required) String This is to specify the color of all the cards that are going to be created
conatinerColor (required) String This is to specify the color of container that contains all the colors

CardList Methods

// Used to create a general card in the card list
  addGeneralCard(id, title, subtitle, imgSrc)

This method creates a General Card with the given properties in the CardList. The method should strictly follow the following property requirements and the properties of makeGeneralCard method in Card Methods:

Property Requirement Type Description
id (required) String The id of the card that you want to create, it will be used to access this card. 
// Used to create a project card in the card list
  addProductCard(id, title, desc, imgSrc, button1, button2)

This method creates a Product Card with the given properties in the cardList. The method should strictly provide the id as shown in the addGeneralCard method. You also need to follow the property requirement of the makeProjectCard method in Card Methods.

// Used to create a teammate card in the card list
  addTeammateCard(id, title, subtitle, imgSrc, links)

This method creates a Teammate Card with the given properties in the cardList. The method should strictly provide the id as shown in the addGeneralCard method. You also need to follow the property requirement of the makeTeammateCard method in Card Methods.

// Used to create a project card in the card list
  addProjectCard(id, title, desc, links)

This method creates a Project Card with the given properties in the cardList. The method should strictly provide the id as shown in the addGeneralCard method. You also need to follow the property requirement of the makeProjectCard method in Card Methods.

// Gets a specific card in the cardList with an id
  getCard(id)

This method returns the card in cardList which matches that id provided in the parameter.

Property Requirement Type Description
id (required) String The id of the card that you want to access. 
// Used to remove a card in the cardList with a perticular id
  removeCard(id)

Deletes the specific card from the cardList and adjusts the DOM automatically.

Property Requirement Type Description
id (required) String The id of the card that you want to delete. 
// Used to add the zoom feature to all the cards in the cardList
  addZoomToAll()

Adds a zoom feature to all the current cards. It also updates the new cards created to have the zoom feature.
